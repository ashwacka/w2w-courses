# Write2Win website

Standalone website for Write2Win online courses. Same design and content as the Expo web app, built with Vite + React + TypeScript.

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Then open the URL shown (e.g. http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/`. Preview the production build with:

```bash
npm run preview
```

## Structure

- `src/theme.ts` – Write2Win colours
- `src/courses.ts` – Course catalog and helpers
- `src/components/` – Header, Layout
- `src/pages/` – Home, Courses, Course detail, Contact
